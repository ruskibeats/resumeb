## 3) Positioning Guidance by Role Type

### A) If applying to Infrastructure Programme Director roles
Prioritise:
- Network + data centre + EUC transformation language
- Delivery-at-scale evidence
- Timelines, multi-site scope, risk/cutover outcomes

De-emphasise:
- Broad COO claims unless directly relevant to advert

### B) If applying to Technical Operations Director roles
Prioritise:
- Operational control language (SLA, service continuity, contract governance)
- Vendor/commercial lifecycle ownership
- Executive/board reporting and governance cadence

### C) If applying to Assurance / Recovery roles
Prioritise:
- Discovery diagnostics, root-cause analysis, remediation plans
- Governance reset and delivery model redesign
- Stakeholder confidence rebuild narratives

## 4) UK CV Convention Notes for Teammates

- Keep CV to **2 pages** where possible (contract market can allow dense but must stay scan-friendly).
- Use a **clear headline + 6–10 core skills** near top.
- Add a **Selected Outcomes** block with quantified or scaled evidence.
- Ensure **dates and concurrent contracts are explicitly labelled** (e.g., “Concurrent advisory engagement”).
- Keep bullets outcome-led: action + scope + result/risk impact.
- Include security clearances only where commercially useful and valid.

## 6) Application Tailoring Checklist (quick use)

Before sending each CV/application:

1. Mirror the advert’s top 8–12 keywords in profile + skills + most recent roles.
2. Reorder achievements to match employer priorities (delivery, operations, assurance).
3. Ensure at least 3 bullets include hard scope signals (sites, regions, endpoints, value, timelines).
4. Keep clearance references accurate and relevant.
5. Remove duplicated narrative and ensure one coherent positioning lane per submission.
